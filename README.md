* [Typescript](https://www.typescriptlang.org/)

## How to use

#### Clone this repository
```bash
git clone https://github.com/fredcoutinho52/microsoft-ui-clone.git
```

#### Install the dependencies
```bash
yarn install
```

##### Run the app
```bash
yarn start
```
